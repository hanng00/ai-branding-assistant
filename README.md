# Fullstack AI Branding Assistant

This projects was used to enhance my skills as a Data engineer.
It is inspired from the following [repo](https://raw.githubusercontent.com/pixegami/copykitt-tutorial/main/README.md)

## Tech Stack

#### Backend Stack

| Type      | Tech                                                                                                     |
| --------- | -------------------------------------------------------------------------------------------------------- |
| Language  | [Python](https://www.python.org/)                                                                        |
| Framework | [FastAPI](https://fastapi.tiangolo.com/)                                                                 |
| Hosting   | [AWS](https://aws.amazon.com/) (Lambda + API Gateway)                                                    |
| Other     | [OpenAI](https://openai.com/), [AWS CDK](https://aws.amazon.com/cdk/), [Docker](https://www.docker.com/) |

#### Frontend Stack

| Type      | Tech                                                          |
| --------- | ------------------------------------------------------------- |
| Language  | [Typescript](https://www.typescriptlang.org/)                 |
| Framework | [NextJS](https://nextjs.org/) / [React](https://reactjs.org/) |
| Styling   | [TailwindCSS](https://tailwindcss.com/)                       |
| Hosting   | [Vercel](https://vercel.com)                                  |
